pointer เป็นตัวแปรที่ใช้เก็บค่าตำแหน่ง หรือ address ใน memory ซึ่งค่าของ address คือจำนวนเต็มบวกที่เริ่มจาก 0 

pointer operators 
    1.& เป็น operator ที่ใช้หาตำแหน่งใน memory ของ operand
    2.* ถูกใช้ในการเข้าถึงข้อมูลที่ถูกอ้างอิงโดยค่าตำแหน่งใน memory ที่ค่าเป็น operand

scanf() เป็นการรับค่าข้อมูลเข้าโปรแกรมจาก standard input(ex. คีย์บอร์ด)
