lined list คือ  โครงสร้างที่ถูกออกแบบมาเพื่อเก็บข้อมูล ข้อมูลแต่ละตัวจะเชื่อมเป็นสาย เพื่อให้ผู้ใช้เข้าถึงได้แต่ละตัว
structure เป็นการกำหนดรูปแบบหรือโครงสร้างข้อมูลที่รวมตัวแปรที่เก็บข้อมูลหลายอย่างที่สัมพันธ์กัน
เราสามารถประกาศและกำหนดค่าเริ่มต้นให้ตัวแปรที่มีโครงสร้าง sturture ได้
self-referential structure ออกแบบมาเพื่อจัดการ linked list โดยสร้างขึ้นจากการใช้ struct ภายในจะมี pointer เพื่ออ้างอิงกลับไปที่ struct
struct node เก็บข้อมูล 2 ส่วน คือ ส่วนที่ใช้เก็บข้อมูล และส่วนที่เป็นตัวแปร pointer 
การเพิ่ม node เข้าไปใน list ทำได้หลายแบบ โดยพิจารณาจากตำแหน่งใน list ที่ต้องการเพิ่ม node เข้าไป
double linked list คือ ลิงก์อันนึงเชื่อมไปโหนดถัดไป แลอีกลิงก์เชื่อมไปโหนดก่อนหน้า
