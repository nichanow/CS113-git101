อาเรย์ คือ กลุ่มของตัวแปรที่ถูกจัดการร่วมกัน ตัวแปรต้องเป็นประเภทเดียวกัน 
เบอร์ของตัวแปรในอาเรย์เริ่มจาก 0 

การใช้งาน
    ขนาดของอาเรย์ตอนประกาศควรเป็นค่าคงที่
    ใช้ #define directive ช่วย หรือ const คีย์เวิร์ด
#define เป็นคำสั่งที่ทำให้ compiler เปลี่ยนอักษรทั้งหมดในโปรแกรมก่อนคอมไพล์ 
const เป็นคีย์เวิร์ดที่บังคับว่าตัวแปรนี้จะไม่มีการเปลี่ยนแปลงตลอดการทำงานของโปรแกรม
ในภาษาซี ไม่มีการเช็คขนาดของอาร์เรย์ให้
การส่งค่าจากอาร์เรย์ไปยังฟังก์ชัน เป็นการ pass by value 

pass by value (ส่งค่าไปฟังก์ชัน ใช้งานได้เลย)
    เป็นการก็อปปี้ค่า สร้างตัวแปร local varible ใหม่ และการเปลี่ยนแปลงค่าในฟังก์ชันจะไม่ส่งผลถึงตัวแปรต้นแบบ 
pass by reference (ส่งตำแหน่งไปที่ฟังก์ชัน)
    ส่งตำแหน่งของตัวแปรจากผู้เรียกไปที่ฟังก์ชัน ตัวแปรในฟังก์ชันเป็นตัวเดียวผู้เรียก และ การเปลี่ยนแปลงค่าในฟังก์ชันจะเปลี่ยนตัวแปรต้นแบบด้วย

2D array
    int a[i][j];
    ซึ่ง i คือแถวแนวนอน
    j คือแถวแนวตั้ง
